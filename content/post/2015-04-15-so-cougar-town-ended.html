So Cougar Town ended, and I don’t know how I feel about that.

Well, trakt knows hout they feel about it.

library(tRakt)
library(ggplot2)

# Get data only if not already in cache
if (file_not_cached(cache_path, cougars)) {
  cougars <- trakt.seasons.summary(
    "cougar-town", extended = "full", episodes = TRUE
  ) %>%
  pull(episodes) %>%
  bind_rows() %>%
  mutate(
    season = as.character(season),
    episode_abs = seq_along(first_aired)
  )
  
  cache_file(cache_path, cougars)
}
# Otherwise read data from cache
cougars <- read_cache_file(cache_path, cougars)

ggplot(data = cougars, aes(x = episode_abs, y = rating, 
                           fill = season, color = season)) +
  geom_smooth(method = lm, se = F) +
  geom_point(shape = 21, color = "black", size = 2) +
  scale_x_continuous(breaks = seq(0, 150, 10), minor_breaks = seq(0, 150, 5)) +
  scale_y_continuous(breaks = seq(0, 10, .5), minor_breaks = seq(0, 10, .1),
                     limits = c(7, 8.5)) +
  scale_color_brewer(palette = "Blues", guide = FALSE) +
  scale_fill_brewer(palette = "Blues") +
  labs(title = "Cougar Town Episode Overview",
       subtitle = "Episode Ratings on trakt.tv",
       x = "Absolute Episode #", y = "Rating (1-10)",
       fill = "Season",
       caption = paste("Data as of", cache_date("cougars.rds", cache_path))) +
  theme(legend.position = "bottom")
ggplot(cougars, aes(x = season, y = rating, fill = season, color = season)) +
  geom_boxplot(alpha = .85, color = "black") +
  scale_y_continuous(breaks = seq(0, 10, .5), minor_breaks = seq(0, 10, .1),
                     limits = c(7, 8.5)) +
  scale_fill_brewer(palette = "Blues", guide = FALSE) +
  labs(
    title = "Cougar Town Episode Overview",
    subtitle = "Episode Ratings on trakt.tv",
    x = "Season #", y = "Rating (1-10)",
    fill = "", color = "",
    caption = paste("Data as of", cache_date("cougars.rds", cache_path))
  )

Six seasons can be a very long time, and not every shows handles the ageing process well. Cougar Town was not one of those shows in my opinion, which is largely due to the show’s self awareness and continued jokes about exactly that. The last season was a rough decline, with Bobby leaving and all, but at least we got a few more bromance jokes out of that, soo… yay?

Anyway, while I can’t even remember the early seasons, I enjoyed the show while it lasted. Not that I was totally excited about every new episode, but it was always nice coming back to it.
